Default Endpoints

Control whether built-in default OSC endpoints are enabled or disabled.

What are Default Endpoints?

Default endpoints are built-in OSC routes that automatically send messages to standard paths based on the platform and message type:

  • YouTube: /onecomme/youtube/comment, /onecomme/youtube/super
  • Bilibili: /onecomme/bilibili/comment, /onecomme/bilibili/gift
  • Niconico: /onecomme/niconico/comment

⚠️ Important:

Disabling default endpoints means only your custom routing rules will send OSC messages. This gives you complete control over message routing but requires you to create custom rules for all desired message types.
