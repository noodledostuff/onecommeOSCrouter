## Image Guidelines

- **Format:** PNG preferred (JPEG for photos)
- **Resolution:** 1920x1080 recommended, minimum 1280x720
- **File Size:** Keep under 500KB per image (use compression if needed)
- **Content:** Capture clear, readable interface elements
- **Annotations:** Add red boxes or arrows to highlight important areas (optional)
- **Privacy:** Remove any personal information (usernames, API keys, etc.)

## Taking Screenshots

### Windows
- Press `Win + Shift + S` for Snipping Tool
- Or use `Alt + PrtScn` for active window

### How to Use
1. Take screenshots following the guide steps
2. Save them with the exact filenames listed above
3. Place them in this directory
4. The website will automatically load them
